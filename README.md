# 🍕 Сейчастье — Система доставки еды v2.0

Полноценный сайт доставки еды с панелью администратора, базой данных SQLite и Telegram-уведомлениями.

## 📁 Структура

```
project/
├── server/app.py        # Flask API + JWT-аутентификация + SQLite
├── public/
│   ├── index.html       # Фронтенд SPA
│   ├── app.js           # Логика (исправленная + расширенная)
│   ├── style.css        # Стили + новые компоненты
│   └── manifest.json    # PWA
├── data/seychasye.db    # SQLite (создаётся автоматически)
├── start.sh             # Быстрый запуск
└── .env.example         # Пример конфигурации
```

## 🚀 Запуск

```bash
pip install flask pyjwt --break-system-packages
bash start.sh
# → http://localhost:5000
# Вход: admin / admin123 (смените сразу!)
```

## 📱 Telegram-уведомления о заказах

```bash
TG_BOT_TOKEN=<токен_от_@BotFather> TG_CHAT_ID=<ваш_chat_id> bash start.sh
```

## 🔐 Безопасность

- Пароли: PBKDF2-SHA256, 260 000 итераций, уникальная соль
- Сессия: httpOnly cookie, истекает через 60 минут
- Брутфорс: 5 попыток → блокировка IP на 5 мин (в БД)
- Серверная валидация цен при оформлении заказа
